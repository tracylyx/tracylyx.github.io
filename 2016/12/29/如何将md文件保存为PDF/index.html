md文件和PDF

一、如何将本地 md 文件在浏览器上展示

  • 方法一：使用 stackedit

  • 方法二：使用 马克飞象

这两种方法都需要你将本地写好的文档复制黏贴到浏览器的编辑框中，立刻就生效看到效果

  • 方法三：下载相应的软件，我在 windows 使用过 markdownPad。也推荐使用有道笔记，它里边可以新增 md 文件，这种方式下，你在本地编辑的同时就能立马看到效果

  • 方法四：使用 sublime 编辑的 md 文档，你可以安装一个插件：markdown preview。使用的时候，shift + command + p，然后输入 markdown preview，接着回车，出现两个选项，“GitHub” 和 “markdown”，任选一个回车，就会帮你自动转到浏览器

二、保存为pdf

使用 markdown 语法写文档，简洁、方便。
那么如何将所谓的 md 文件转为 PDF 呢？

现在来为你快速揭秘：

  1. 将 md 文件在浏览器（这里以谷歌浏览器为例）打开浏览

  2. Mac 下，command + p 打开打印框框，左侧如果看不到的“保存”按钮，那么就要选择“更改”，选中“另存为PDF”
    这样，以后 command + p 都可以直接点保存啦～～

😄😄😄，就是这么简单～～
